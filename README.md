# OCB MicroSplat Roads Demo Mod - Addon for OCB MicroSplat

Spruce up vanilla road (gravel/tar) by using new textures.

Requires [OcbMicroSplat][1] (Download from [Nexusmods][3]).

You need to disable EAC to use this mod!
